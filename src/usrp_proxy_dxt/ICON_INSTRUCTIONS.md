# USRP MCP Proxy Icon

Since this is a text-based environment, I'll provide instructions for creating an appropriate icon:

## Icon Requirements (DXT Specification)
- Format: PNG
- Size: 64x64 pixels (recommended)
- Design: Should represent SDR/USRP/Radio functionality

## Suggested Icon Design
- Base: Radio wave symbol (concentric circles)
- Color scheme: Blue/green tech colors
- Add small proxy/connection indicator (arrows or chain link)
- Text: "USRP" or "SDR" if space allows

## To create the icon:
1. Use any image editor (GIMP, Photoshop, Canva, etc.)
2. Create a 64x64 PNG image
3. Use radio wave symbols or RF-related imagery
4. Save as `icon.png` in this directory
5. Update manifest.json to include: `"icon": "icon.png"`

## Alternative
If you have an existing UHD or USRP logo, resize it to 64x64 and use that.
