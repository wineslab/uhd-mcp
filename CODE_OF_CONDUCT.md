# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1, as its Code of Conduct.

In short: we are committed to providing a friendly, safe, and welcoming environment for everyone.
Be respectful, assume good intent, and keep discussions constructive. Harassment or
discrimination of any kind is not tolerated.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

To report unacceptable behavior, contact the maintainer at **a.lacava@northeastern.edu**.
All reports will be reviewed and handled confidentially.
